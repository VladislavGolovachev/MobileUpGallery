# MobileUpGallery
Это тестовое задание для компании MobileUp. Приложение позволяет авторизироваться пользователю в ВК. После авторизации будет показана сетка медиа группы. Пользователь может выбрать раздел фото или видео (по требованию представлено в высоком качестве). При нажатии на фото пользователь попадает на экран с фото во весь экран. Предусмотрена возможность поделиться, скопировать и сохранить в галерею фото (при удачном и неудачном сохранении выводится алерт). При нажатии на видео открывается экран с видео во весь экран. Также предусмотрена возможность поделиться и скопировать ссылку на видео. Присутствует возможность выйти из аккаунта.  
При повторном заходе в приложение, в случае, если токен еще действителен, то появляется сразу сетка с медиа. В ином случае — экран авторизации.  
Тестовое задание необходимо было декомпозировать, поэтому приложена таблица подзадач и оценка времени на их выполнение.

## Особенности
* UICollectionView с кастомными ячейками,
* UICollectionViewDataSourcePrefetching,
* Keychain для хранения токена пользователя,
* NSCache для хранения медиа группы (предусмотрен случай, когда приложение уходит в background и cache очищается),
* UIActivityViewController,
* обработка ошибок по всему проекту с выводом алерта (Keychain, ошибки сети и ошибки, полученные с сервера ВК),
* кастомный дизайн (узнал про SFSymbols),
* кастомный SegmentedControl (столкнулся с проблемой, что SegmentedControl от apple нельзя было кастомизировать под тестовое задание),
* кастомный PaddingLabel,
* добавлена русская локализация,
* GCD,
* WebKit,
* паттерн builder для сборки модулей,
* разделение проекта на слои.
